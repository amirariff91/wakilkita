# Impeccable Design Context — WakilKita

## Project
WakilKita is a trust-first Malaysian civic mandate MVP. It is not an online election and not an SPR replacement. The product helps verified constituents nominate and support the local representative they want, then compares those reps against constituency issues and open data.

## Emotional goal
Serious, civic, calm, accountable. It should feel like public infrastructure: credible enough for NGOs/media, simple enough for normal residents, and careful about privacy.

## Visual system
- Palette: deep ink #071613, civic green #0F6B4D, soft mint #DDF7E8, warm parchment #F7F0E4, audit amber #D99A1E, slate #33413D.
- Typography: editorial serif-style headings via Georgia fallback; clean system sans body.
- Shape: rounded civic panels, map/grid motifs, seal-like counters, transparent audit cards.
- Motion: minimal; trust-sensitive UI should not feel gimmicky.

## Anti-patterns to avoid
- No generic startup hero with vague gradients.
- No crypto/blockchain theatre.
- No “official election” language.
- No decorative dashboards without clear trust meaning.
- No individual political preference exposure.
- No thick left-accent side-tab cards, nested card piles, or template feature grids.
